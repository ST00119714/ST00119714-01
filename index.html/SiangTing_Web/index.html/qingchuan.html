晴川居

光影之間，家的溫度流動。
鄰英明國中靜巷，靜中帶靜、鬧中取靜，以人文氣息與自然採光為設計核心。 近捷運輕軌、交通便捷，串聯城市與自然的日常，讓幸福自在流動於每一扇窗前。

建案介紹

翔廷建設全新力作【晴川居】，鄰近英明國中與學區綠意，規劃精緻 2～3 房格局， 採用制震結構與節能設計，兼顧居住安全與舒適。 生活圈完善、近輕軌捷運、商圈機能齊備， 在城心與自然之間，築起一座恬靜而深厚的家園。
